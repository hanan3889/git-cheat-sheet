Git Cheat sheet

Qu'est ce que Git ?

Git est le système de contrôle de version distribué, gratuit et open source, le plus utilisé.
À l'origine, il a été développé en 2005 par Linus Torvalds, le créateur bien connu du noyau du système d'exploitation Linux.
Ce Git Cheat Sheetvous présente les commandes Git les plus importantes et les plus utilisées.
